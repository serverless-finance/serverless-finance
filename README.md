# serverless-finance

AWS serverless finance management.

## Requirements

AWS CDK:

```shell
npm i -g cdk
```

## Getting started

Install dependencies

```shell
npm i
```

Bootstrap CDK on Account (only required once)

```shell
cdk bootstrap
```

Deploy application

```shell
cdk deploy
```

## Cleanup

Delete all resources deployed by CDK

```shell
cdk destroy
```
